Домашнее задание Git
1. Установите и настройте Git
2. Создайте учетную запись BitBucket
3. Создайте репозиторий для вашего проекта на BitBucket
4. Создайте Maven Java проект с одним классом, который печатает в консоль сообщение Hello World
5. Добавьте к проекту гит репозиторий
6. Закомитьте состояние проекта
7. Привяжите локальный репозиторий к созданному удаленному
8. Отправьте проект в удаленный гит репозиторий
9. В отдельной локальной ветке добавьте вывод второго сообщения Hello World в консоль и отправьте ее в удаленный гит репозиторий
10. Через Pull Request влейте вашу доработку в master ветку
11. Обновите локальную master ветку

Как проверить что все правильно? Вы прошли все шаги без ошибок. В итоге в вашей локальной и удаленной master ветке по два коммита и их состояние идентично

Что делать если что-то пошло не так? Можете повторно просматривать практическую часть урока, и параллельно повторять действия преподавателя, корректируя их под решение домашнего задания
