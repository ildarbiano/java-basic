Задание
Домашнее задание Maven
1. Создайте Java проект с поддержкой Maven
2. Укажите используемую версию JDK
3. Укажите корректный идентификатор вашего Maven проекта учитывая, что вы создаете проект компании StepUp, он входит в группу проектов платежей payments, а сам проект связан с мобильными платежами и называется Mobile Pay
4. Добавьте к проекту в библиотеки JUnit 4.13.2, Apache Commons Lang 3.12.0
5. Укажите что в результате сборки приложения исполняемый файл должен называться mobile-pay.jar6. Создайте MainApplication класс с методом main(), который печатает в консоль сообщение Welcome to Mobile Pay
7. Выполните сборку проекта

8. (  не обязательная часть ) В jar архив должен попасть файл info.txt с сообщением Hello World (то есть файл, который вы заранее подготовили, не надо его формировать как-то кодом). Проверьте что проект запускается из консолитерминала ОС
