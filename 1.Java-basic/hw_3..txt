for				- цикл.
int i = 0 			— инициализация счетчика. Создаем переменную i типа int. Начальное значение 0 (индексы в Java начинаются с 0). Переменная i существует только внутри цикла.
i < args.length 		— условие продолжения args.length — длина массива аргументов. Цикл будет выполняться, пока i меньше длины массива. Например, если передано 3 аргумента, цикл выполнится для i = 0, 1, 2.
i++ 				— инкремент (увеличение на 1). После каждой итерации i увеличивается на 1. i++ эквивалентно i = i + 1.
args = ["Java", "Is", "Awesome"]
args.length = 3
Итерации:
1. i=0: 0 < 3? ДА → печатаем args[0] = "Java"
2. i=1: 1 < 3? ДА → печатаем args[1] = "Is"  
3. i=2: 2 < 3? ДА → печатаем args[2] = "Awesome"
4. i=3: 3 < 3? НЕТ → цикл завершается

public class My1Class_3 {                           //описание класса с именем
    public static void main(String... args) {       //точка входа, метод main, с жёстко заданной сигнатурой.
            System.out.println("Hello World");      //функция класса PrintStream, по статическим полем out класса System
            for (int i = 0; i < args.length; i++) { //
                System.out.println(args[i] + " - " + "changed to = " + args[i+1]);
        }
    }
}

    System.out.println(args[i] + " - " + "changed to = " + args[i+1]);
